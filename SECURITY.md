# Security

If you find a vulnerability — in the MCP server, the consumer CLI, or anything
this repo ships — please report it privately rather than opening a public issue:

- **GitHub**: use [private vulnerability reporting](https://github.com/whiffernet/embeddington/security/advisories/new)
  on this repository.

You'll get an acknowledgement within a few days. This is a personal project;
there is no bounty, but reports are genuinely appreciated and acted on.

The graph data itself is derived from public ServiceNow documentation — see the
provenance section in the README. If you believe the _data_ contains something
it shouldn't, the same channel works.
